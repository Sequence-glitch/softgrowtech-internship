📌 Project Overview

The goal of this project was to build a resume-style website using semantic HTML elements without relying on CSS or JavaScript.

The page showcases:

 • Technical and soft skills
 • Featured projects
 • Contact information

This demonstrates how structured HTML alone can present professional information effectively.
