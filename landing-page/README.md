# 🌐 SoftGrowTech – Responsive Landing Page

A modern and fully responsive landing page built for SoftGrowTech, an IT & Digital Solutions organization focused on empowering students and businesses with real-world technology skills.

This project was built using only HTML5 and CSS3, following semantic structure and responsive design principles.

---

## 📌 Project Overview

This landing page highlights:

- Company Overview
- Mission & Vision
- Services & Programs
- Internship Call-to-Action
- Contact & Footer Information

The design focuses on clarity, professionalism, and responsiveness across all screen sizes.

---

## 🚀 Features

- ✅ Semantic HTML structure (`header`, section, `footer`)
- ✅ Responsive layout using Flexbox & CSS Grid
- ✅ Sticky navigation bar
- ✅ Hero section with background image + overlay
- ✅ Smooth scroll behavior
- ✅ Mobile-first responsive media queries
- ✅ Modern button styles with hover effects
- ✅ Clean and professional UI

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

---

## 📱 Responsiveness

The layout adapts to:

- 💻 Desktop screens
- 📱 Tablets
- 📲 Mobile devices (768px and below)

Navigation stacks vertically on smaller screens, buttons adjust width, and sections scale proportionally.
